## HC_Challenge_Django

### API Endpoints

* **/users** (Get users)
* **/reports** (Get reports)

### Install 

    pip install -r requirements.txt

### Load data

    python manage.py loaddata users
    python manage.py loaddata reports
    python manage.py loaddata report_responses

### Generate data

    python manage.py seed

### Usage

    python manage.py runserver
